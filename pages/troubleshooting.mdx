# 困ったときの対処法

この章ではバイオインフォマティクスの解析を行う際に困ることの対処法について、できるだけ応用可能な形で紹介する。

## 公式ドキュメントを読む
各解析ソフトウェアには `man` ページや `--help` オプションのヘルプメッセージ、PDF マニュアル、ドキュメンテーションなどが存在します。まずはそれらを読むことが重要です。

## フォーラムを活用する
以下のようなフォーラムが利用可能です。
- 解析ソフトウェアのメーリングリスト
  - Google Group などで公開されている場合があります。
- BioStars (Q&Aサイト)
- 解析ソフトウェアの GitHub Issue
  - このような場所では解析ソフトウェアの著者が直接質問に答えてくれることがあります。しかし、そのためには上手な質問の仕方や礼儀が必要です。
  - 例) SAMtools の GitHub Issue https://github.com/samtools/samtools/issues
